# ==============================================================================
# TOPAS Monte Carlo Simulation: DNP Target Cold Irradiation Study
# Jefferson Lab Upgraded Injector Test Facility (UITF)
# ==============================================================================
#
# PURPOSE:
#   Simulate the 8 MeV electron beam irradiation of solid ammonia (ND3/NH3)
#   and deuterated butanol targets in a variable-temperature cryostat at the
#   UITF. Score DoseToMedium throughout target volumes, characterize the
#   longitudinal dose falloff, determine optimal beam-spreading aluminum window
#   thickness, and predict the dose uniformity and deposited energy as a
#   function of irradiation depth for both warm (87 K) and cold (2-4 K)
#   irradiation configurations.
#
# PHYSICS MODEL:
#   - Full electromagnetic cascade (Geant4 Livermore + G4EmStandard_opt4)
#   - Accurate electron stopping powers in cryogenic ammonia phases
#   - Multiple scattering via Urban model (tuned for sub-cm targets)
#   - Bremsstrahlung, Moller scattering, ionization explicitly tracked
#
# GEOMETRY (real dimensions from proposal):
#   - Beam spreader: Al window (variable thickness, default 0.5 mm)
#   - Cryostat entrance window: 0.1 mm Al (upstream vacuum barrier)
#   - Cryogenic liquid bath: LHe or LAr depending on phase (toggleable)
#   - Target sample holder: Hexagonal cage, 6.5 cm length, 1.42 cm radius
#   - Target material: ND3 or NH3 spherical beads (~1 mm radius)
#     represented as a uniform slab for dosimetry + a bead geometry branch
#   - Beam interaction area from proposal: 6.42 cm^2 (r ~ 14.3 mm rastered)
#
# KEY OBSERVABLES:
#   1. DoseToMedium (Gy) vs. depth in ND3/NH3 slab
#   2. Fluence (e-/cm2) map — correlated to proposal dose targets:
#        Warm dose: 1e17 e-/cm2
#        Cold dose: 5e15 e-/cm2
#   3. Lateral dose uniformity across 14 mm rastered beam radius
#   4. Energy deposition in Al windows (heat load estimate for cryostat)
#   5. Bremsstrahlung photon spectrum exiting downstream (shielding design)
#   6. Backscatter fraction (relevant to front vs. back irradiation geometry)
#
# ADVANCED PREDICTION FEATURES:
#   - Parametric scoring: DoseToMedium scored in 1 mm depth bins (slab) and
#     in individual 1 mm radius spherical bead geometry (realistic geometry)
#   - Electron fluence spectrum at target entrance and exit faces
#   - Secondary particle production: photons, neutrons (via Bremsstrahlung)
#   - Linear energy transfer (LET) spectrum for radical formation modeling
#   - Range-energy relationship verification at 8 MeV in cryogenic ND3
#   - Front vs. back irradiation symmetry check
#   - Dose-volume histogram (DVH) approximation via radial dose binning
#
# REFERENCE CONDITIONS (from proposal):
#   Beam energy/current: 8 MeV, 1-10 uA, rastered to ~14 mm radius
#   Warm irradiation: 10 uA, 87 K (LAr bath), 8 MeV electrons
#   Cold irradiation:  1 uA,  2-4 K (LHe bath), 8 MeV electrons
#   Target dose (warm): ~1e17 e-/cm2; 2.85 hr/side, front + back
#   Target dose (cold): ~5e15 e-/cm2; 1.42 hr/side, front + back
#   Batch mass: 250 mg material in a variable-temperature cryostat holder
#   Holder geometry: cylinder/hex cage envelope, L = 6.5 cm, R = 1.42 cm
#   Material density (solid ND3 at 2 K): ~1.007 g/cm3
#   Material density (solid NH3 at 87 K): ~0.730 g/cm3 (liquid-like phase)
# AUTHOR: SAMUEL TAKWIRIRA
# COLLABORATORS: UNH DNP Group / Jefferson Lab Target Group collaboration
# DATE: May 2026
# TOPAS VERSION: 3.9+ (Geant4 10.7.x backend required)
# ==============================================================================

# ==============================================================================
# SECTION 7 — SCORING: LINEAR ENERGY TRANSFER (LET)
# ==============================================================================
#
# The LET spectrum directly governs radical (free radical) yield through
# the G-value relationship: G(radical) = f(LET).
# For ND3: Ṅ D2 radicals dominate at low LET; atomic N appears at high LET.
# Scoring LET allows us to predict the spatial distribution of radical types,
# which underlies why cold irradiation changes the ND3 polarization — the
# temperature-dependent diffusion radius of radicals created at low LET
# differs at 2 K vs. 77 K, affecting DNP transfer efficiency.
#
# We score both track-averaged LET (LETt) and dose-averaged LET (LETd).
# LETd is the physically relevant quantity for biological/radical systems:
#   LETd = integral(LET^2 * Phi(LET) dLET) / integral(LET * Phi(LET) dLET)

# NOTE: TOPAS 3.9 has no generic LET scorer for electrons. ProtonLET is proton-specific
# and not valid for this 8 MeV electron beam. LET is reconstructed in post-processing:
#   LETd (keV/um) = EnergyDeposit_mean / (Fluence_mean * VoxelVolume)
# where VoxelVolume = pi * Rmax^2 / nR * HL*2 / nZ (per voxel in TargetSlab2D).

# --- 7A: EnergyDeposit depth profile (numerator for LETd reconstruction) ---

s:Sc/LETd_Depth/IfOutputFileAlreadyExists = "Overwrite"

i:Sc/LETd_Depth/ZBins                     = 65
b:Sc/LETd_Depth/ReportMean    = "True"
b:Sc/LETd_Depth/ReportVariance = "True"

# --- 7B: Fluence depth profile (denominator for LETt reconstruction) ---

# ==============================================================================
# SECTION 11 — SCORING: DOSE RATE AND HEAT LOAD CONVERSION
# ==============================================================================
#
# Converts Monte Carlo DoseToMedium (Gy per primary) to absolute dose rate
# (Gy/s) using the actual beam current (I) and electron charge (e):
#   dose_rate (Gy/s) = DoseToMedium_per_primary (Gy) * I (A) / e (C)
#   heat_load (W)    = DoseToMedium_per_primary (Gy/kg) * mass (kg) * I/e
#
# For warm irradiation:
#   I = 10 uA → N_e/s = 6.242e13 e-/s
#   mass = 0.25 g (250 mg) = 2.5e-4 kg
#
# For cold irradiation:
#   I = 1 uA → N_e/s = 6.242e12 e-/s
#   Heat load constraint: LHe refrigerator capacity ~ 0.1–0.5 W
#   Must verify deposited power < refrigerator capacity at 2 K
#
# These conversion factors are written as TOPAS parameters for clarity:

u:Sc/Conversion/WarmCurrentA          = 10.0e-6           # beam current in Amperes (10 uA)
u:Sc/Conversion/ColdCurrentA          = 1.0e-6            # beam current in Amperes (1 uA)
u:Sc/Conversion/ElectronChargeC       = 1.602e-19         # electron charge in Coulombs
u:Sc/Conversion/WarmElectronsPerSec   = 6.242e13          # electrons/s = 10 uA / e
u:Sc/Conversion/ColdElectronsPerSec   = 6.242e12          # electrons/s = 1 uA / e
d:Sc/Conversion/TargetMass            = 0.25 g            # 250 mg target mass

# NOTE TO USER: After simulation, compute:
#   AbsoluteDoseRate = Sc/DoseDepth/Mean[i] * Sc/Conversion/WarmElectronsPerSec
#   HeatLoadW        = AbsoluteDoseRate * Sc/Conversion/TargetMass
# Compare HeatLoadW against cryostat refrigerator capacity to confirm feasibility.
